Dieses ist eine Übungsanwendung.

Sie bietet eine Stammdatenverwaltung von Kunden.

Die Benutzeroberfläche der Anwendung kann die Sprachen Deutsch oder Englisch benutzen. Die Sprache kann in der Datei rbsjava.ini konfiguriert werden. Sie kann auch im laufenden Betrieb geändert werden.

Die Kundendaten sind in einer "Datenbank" gespeichert, welche derzeit als einfache Textdatei im lokalen Filesystem existiert, deren Name wiederum in rbsjava.ini konfiguriert ist. Die verwendete Datenbank kann ebenfalls im laufenden Betrieb geändert werden.

Änderungen der aktuellen Sprache und der aktuellen Datenbank sind nur wirksam, bis die Anwendung endet; sie können jedoch auch dauerhaft von der Anwendung in rbsjava.ini gespeichert werden.
